Ejercicio 1):

3)Ingresar por teclado dos valores y mostrar el resultado de su adición

Resolución:

import java.util.Scanner;
public class suma {
public static void main(String[] args){

int valor1=0;
int valor2=0;
int suma=0;
Scanner teclado=newScanner(System.in);
System.out.print("ingrese valor 1");
valor1=teclado.nextInt();
System.out.print("ingrese valor 2");
valor2=teclado.nextInt();
suma=valor1+valor2;
System.out.println("el valor de la suma entre "+valor1 +" y "+ valor2+"es igual a "+ suma);
}
}
