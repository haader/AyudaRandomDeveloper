Ejercicio 1):

1)Ingresar por el teclado el nombre y el apellido, Mostrar Nombre y apellido

Resolución:

import java.util.Scanner;
public class CargaDatos { public static void main(String [] args){

String nombre="";
String apellido="";
Scanner teclado=new Scanner(System.in);

System.out.println("ingese su nombre");

nombre=teclado.next();

System.out.println("ingrese su apellido");

apellido=teclado.next();

System.out.println("Hola "+nombre+" "+apellido);

}
}
