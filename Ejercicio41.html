Ejercicio 4.1):

Dado un entero N entre 1 y 10 (inclusive), la computadora muestra la tabla de multiplicar de N

Resolución:

import java.util.Scanner;
public class Ejercicio41 {

// se debe remplazar por el nombre del ejercicio por el que le has dado


public static void main(String[] args) {

int numero=0;
int resultado=0;
Scanner tecla=new Scanner (System.in);
System.out.println("Ingrese un numero entre 1 y 10 Inclusive: ");
numero=tecla.nextInt();


while(numero>=1&&numero<=10){

for(int i=0;i<=10;i++){
resultado=numero*i;
System.out.println(numero+"X"+i+"="+resultado);
}

System.out.println("ingrese otro numero");
numero=tecla.nextInt();
}
System.out.println("ingresaste un numero fuera del rango");

}
}
